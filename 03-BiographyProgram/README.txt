Biography Program - C++

This program prompts the user to enter their full name, hometown, and age.
It demonstrates proper use of data types and input validation in C++.

Features:
- Accepts full names and hometowns with spaces.
- Validates age input to ensure a number is entered.
- Uses a single 'cout' to display all output as per requirement.

How to Compile and Run:
1. Open a terminal or use an IDE like Code::Blocks or Visual Studio.
2. Compile using: g++ -o biography biography.cpp
3. Run using: ./biography
