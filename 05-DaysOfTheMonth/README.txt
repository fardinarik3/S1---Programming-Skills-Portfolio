Days of the Month - C++ Program

This simple program asks the user to enter a number representing a month (1 = January, ..., 12 = December)
and uses a switch statement to output how many days are in that month.

To use:
1. Compile the code using a C++ compiler.
2. Run the program.
3. Enter a number between 1 and 12 to get the number of days in that month.
