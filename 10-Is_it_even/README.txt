
Is It Even? - C++

This program checks whether an entered number is even or odd.

How it works:
- The user is asked to enter a number in the 'main' function.
- The number is passed to the 'checkEven' function.
- The function returns a message stating whether the number is even or odd.
- The returned message is printed by the 'main' function.

How to Compile and Run:
1. Save the code as is_it_even.cpp
2. Compile: g++ is_it_even.cpp -o is_it_even
3. Run: ./is_it_even
