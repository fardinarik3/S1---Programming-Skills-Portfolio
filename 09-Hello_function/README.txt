Hello Function Program - C++

This is a basic C++ program that demonstrates how to use a user-defined function.

What the Program Does:
----------------------
- Defines a function called `hello` that prints "Hello" to the console.
- Calls the function from `main()` to display the message.

File Included:
--------------
- hello_function.cpp: The source code file.

How to Compile and Run:
------------------------
1. Open the file in a C++ editor.
2. Compile with: g++ hello_function.cpp -o hello
3. Run using: ./hello
