README - Simple Sums Exercise

This project demonstrates a basic C++ program that calculates the sum of two integers.

Steps:
1. Declare and initialize two integers: 8 and 10.
2. Add them and store the result in a third variable.
3. Print the result to the console.

Files included:
- simple_sums.cpp: The C++ source file.
- readme.txt: This file.

How to run:
1. Open the file `simple_sums.cpp` in a C++ editor or IDE.
2. Compile using: g++ simple_sums.cpp -o simple_sums
3. Run the program to see the output.

Expected Output:
The sum is: 18
